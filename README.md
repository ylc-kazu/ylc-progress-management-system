# YLC Progress Management System

YLC Progress Management System は、YLC株式会社が教室運営の効率化と教育品質の向上を目的として開発する、総合的な進捗管理・メンター支援システムです。

本システムは、個別指導型プログラミング教室における以下の課題を解決します。

- メンターの作業負担（記録・共有・文章作成）の軽減
- 保護者共有内容の品質向上（AIによる文章生成）
- 生徒の学習進捗の一元管理（タイピング・カリキュラム・履歴）
- 教室運営の最適化（生徒・メンターの割り振り、教材管理）
- PDFレポートの自動生成と Google Drive への自動アップロード

Spring Boot（Java）を基盤とし、AI文章生成には Python（FastAPI）を採用。  
現場の運用に合わせた、軽量で実用的な教育支援システムを目指します。

## System Overview

YLC Progress Management System は、個別指導型プログラミング教室の現場で発生する
「記録・共有・管理・運営」の業務を一元化し、メンターの作業効率と保護者への情報提供の質を同時に高める教育支援システムです。

### 1. メンター支援
- レッスン記録の入力を最小限に
- 箇条書きからAIが保護者向け文章を自動生成
- PDFレポートを自動作成
- iPadから直感的に操作可能

### 2. 保護者共有の品質向上
- レッスン内容をPDF化
- Google Drive に自動アップロード
- 生徒ごとのフォルダ構造（年度・日付）を自動生成
- 共有内容の統一と精度向上

### 3. 生徒の進捗管理
- タイピング記録の蓄積とグラフ化
- カリキュラム進捗の可視化
- 使用教材・学習履歴の一元管理
- 過去のレッスンをいつでも参照可能

### 4. 教室運営の補助
- 生徒・メンターの割り振り管理
- 教室全体の稼働状況を可視化
- 教材・デバイスの利用状況を記録
- 運営判断のためのデータ基盤を提供
